## Technologies
* React/Redux for frontend
* Rails for backend

## How to setup and run Application
* git clone https://github.com/mkpdev/react-rails-test.git

#### To run frontend application :
* cd `react-front`
* run `yarn install`
* run `yarn start`

#### To run backend application :
* cd `rails-api`
* run `bundle install`
* run `rake db:create && rake db:migrate`
* run `rails s`
